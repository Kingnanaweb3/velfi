# Escrow

Escrow holds money safely until a condition is met — then releases it. It's how you pay for something with confidence that the other side delivers.

### How to set up escrow

> "Hold $2,000 until the work is delivered, then release."

Velfi locks the funds in escrow. The money isn't with you anymore, but it isn't with the recipient yet either — it's held until you release it.

### How it works

1. You set aside the funds in escrow.
2. The other party does the work or delivers the goods.
3. When you're satisfied, you release the funds and they're paid.
4. If something goes wrong, you can cancel before release and the funds return to you.

{% hint style="info" %}
Escrow protects both sides. The recipient knows the money is set aside and real. You know it won't be released until you're happy.
{% endhint %}

### Common uses

* Paying a freelancer for a deliverable.
* Buying or selling between two people who don't know each other.
* Any deal where both sides want assurance before money changes hands.

### Canceling

You can cancel an escrow and reclaim your funds anytime before you've released them.
