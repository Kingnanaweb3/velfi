# Automations

Automations are rules that run on their own. Set them once, and Velfi keeps them going — no need to repeat the action each time.

### Scheduled payments

Pay on a recurring cadence:

> "Send rent to landlord.vel every Friday."

> "Pay everyone on payroll on the 1st of the month."

Velfi runs the payment automatically each time, until you stop it.

### Rule-based automations

Trigger an action based on an event:

> "Move 10% of every payment I receive into savings."

Now, every time money comes in, Velfi automatically sets aside the percentage you chose — without you lifting a finger.

### How automations stay safe

You approve an automation once, when you set it up. From then on it runs within the rules you defined. You can review or cancel any automation at any time.

### Common uses

* Recurring rent, salary, or subscription payments.
* Automatic savings on every inbound payment.
* Regular transfers to a partner, family member, or business account.
